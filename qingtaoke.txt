
itemlist

名称	类型	必填	说明

app_key	String	是	您在轻淘客申请的app_key

v	String	是	当前API版本号（目前是1.0）

page	Int	否	分页获取数据 默认1 最多获取到600页（每页100条）

page_size	Int	否	每页显示多少条数据 , 可选值 [100 , 200] , 默认100

sort	Int	否	
排序方式：              
      1 : 人气排序从高到低（默认）；
      2 : 最新排序从高到低；
      3 : 销量排序从高到低；
      4 : 价格排序从低到高；
      5 : 佣金排序从高到低；
      7 : 价格排序从高到低；
      8 : 人气排序从低到高；
      9 : 最新排序从低到高。

cat	Int	否	
商品分类：              
      0 :   全部（默认）；                          
      2  :  母婴;
      3  :  美妆; 
      4  :  居家; 
      5  :  鞋包配饰; 
      6  :  美食; 
      7  :  文体; 
      8  :  家电数码; 
      9  :  其他; 
      10 :  女装; 
      11 :  内衣; 
      12 :  男装; 

min_price	Int	否	券后价区间（最小值）

max_price	Int	否	券后价区间（最大值）

new	Int	否	是否取今日新品，0表示全部，1表示取今日新品。默认0

is_ju	Int	否	是否取聚划算商品，0表示全部，1表示取聚划算。默认0

is_tqg	Int	否	是否取淘抢购商品，0表示全部，1表示取淘抢购。默认0

is_ali	Int	否	是否取阿里券商品，0表示只取非阿里券商品，1表示只取阿里券商品，不传获取全部。默认获取全部





search



名称	类型	必填	说明

s_type	int	是	1搜索标题和文案 2只搜索标题 3只搜索文案

app_key	String	是	您在轻淘客申请的app_key

v	String	是	当前API版本号（目前是1.0）

key_word	string	是	搜索关键词 支持搜索商品链接（搜索的商品链接必须urlencode编码）和商品ID

page	int	否	分页获取数据 默认1 最多获取到100页（每页100条）

cat	Int	否	
商品分类：              
      0 :   全部（默认）；                          
      2  :  母婴;
      3  :  美妆; 
      4  :  居家; 
      5  :  鞋包配饰; 
      6  :  美食; 
      7  :  文体; 
      8  :  家电数码; 
      9  :  其他; 
      10 :  女装; 
      11 :  内衣; 
      12 :  男装; 

min_price	Int	否	券后价区间（最小值）

max_price	Int	否	券后价区间（最大值）

sort	Int	否	
排序方式：              
      1 : 人气排序从高到低（默认）；
      2 : 最新排序从高到低；
      3 : 销量排序从高到低；
      4 : 价格排序从低到高；
      7 : 价格排序从高到低；
      8 : 人气排序从低到高；
      9 : 最新排序从低到高。

is_ju	Int	否	是否取聚划算商品，0表示取全部，1表示取聚划算。默认0

is_tqg	Int	否	是否取淘抢购商品，0表示取全部，1表示取淘抢购。默认0

is_ali	Int	否	是否取阿里券商品，0表示只取非阿里券商品，1表示只取阿里券商品，不传获取全部。默认获取全部
